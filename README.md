# FlickrFeedClient

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 7.3.9.

## Set-up
- Define the url for server in `src/app/services/config`

## Run
```sh
$ npm install
$ ng serve --port 4200 --open
```

## Build for production
```jshelllanguage
$ ng build --prod
```
